Mithril Aggregator Server (0.1.5)

Download OpenAPI specification:Download

URL: https://github.com/input-output-hk/mithril License: BSD 3-Clause License Terms of Service

The REST API provided by a Mithril Aggregator Node in a Mithril network.

You can play with it here.

Get current epoch settings

Returns the information related to the current epoch:

  • protocol parameters for current epoch
  • protocol parameters for next epoch (to setup cryptography, allowing signers to register)

Responses

https://aggregator.release-preprod.api.mithril.network/aggregator/epoch-settings
https://aggregator.pre-release-preview.api.mithril.network/aggregator/epoch-settings
https://aggregator.testing-preview.api.mithril.network/aggregator/epoch-settings
http://localhost:8080/aggregator/epoch-settings

Response samples

  • 200
  • default
Content type
application/json
{
  • "epoch": 329,
  • "protocol": {
    • "k": 857,
    • "m": 6172,
    • "phi_f": 0.2
    },
  • "next_protocol": {
    • "k": 2422,
    • "m": 20973,
    • "phi_f": 0.2
    }
}

Get current pending certificate information

Returns the information related to the current pending certificate:

  • protocol parameters, for current and next epoch (to setup cryptography)
  • beacon information (where on the chain the pending certificate should be triggered)
  • entity type of the message that must be signed
  • verification keys of the signers, for current and next epoch

Responses

https://aggregator.release-preprod.api.mithril.network/aggregator/certificate-pending
https://aggregator.pre-release-preview.api.mithril.network/aggregator/certificate-pending
https://aggregator.testing-preview.api.mithril.network/aggregator/certificate-pending
http://localhost:8080/aggregator/certificate-pending

Get certificate by hash

Returns the certificate identified by its hash

path Parameters
certificate_hash
required
string <bytes>
Example: AsB0vSvijo8FIfrGFwBtdCNQVhBc9P24qQwKgWw4mr5kRZL

Hash of the certificate to retrieve

Responses

https://aggregator.release-preprod.api.mithril.network/aggregator/certificate/{certificate_hash}
https://aggregator.pre-release-preview.api.mithril.network/aggregator/certificate/{certificate_hash}
https://aggregator.testing-preview.api.mithril.network/aggregator/certificate/{certificate_hash}
http://localhost:8080/aggregator/certificate/{certificate_hash}
