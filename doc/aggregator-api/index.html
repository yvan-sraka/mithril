You can play with it here.

Get current pending certificate information

Returns the information related to the current pending certificate:

  • protocol parameters (to setup cryptography)
  • beacon information (where on the chain the pending certificate should be triggered)
  • verification keys of the signers

Responses

https://aggregator.api.mithril.network/aggregator/certificate-pending
http://localhost:8080/aggregator/certificate-pending

Get certificate by hash

Returns the certificate identified by its hash

path Parameters
certificate_hash
required
string <bytes>
Example: AsB0vSvijo8FIfrGFwBtdCNQVhBc9P24qQwKgWw4mr5kRZL

Hash of the certificate to retrieve

Responses

https://aggregator.api.mithril.network/aggregator/certificate/{certificate_hash}
http://localhost:8080/aggregator/certificate/{certificate_hash}
